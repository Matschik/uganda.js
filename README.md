# TRUMP.JS

MAKE THE WEB GREAT AGAIN

I don't want any of those images that aren't from my server!

[demo](https://bullg.it/trump.js/)

# Usage

include `trump.js` on any willing page like this: 

```html
        <script src="path/to/trump.js"></script>
    </body>
</html>
```
